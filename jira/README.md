# Jira Skill for Claude Code

Claude Code에서 Jira Cloud 티켓을 관리하는 스킬입니다.

## 사용법

명령어를 외울 필요 없이 자연어로 요청하면 됩니다:

```
"현재 브랜치 티켓 보여줘"
"내 담당 티켓 목록 보여줘"
"이 티켓 진행중으로 바꿔줘"
"홍길동님한테 할당해줘"
"작업 완료했다고 코멘트 달아줘"
"이번 스프린트 이슈 뭐있어?"
```

아래 명령어 가이드는 참고용입니다.

## 설치

1. 스킬 폴더 복사:
```bash
cp -r jira ~/.claude/skills/
```

2. 스킬 설정:
```bash
/jira 설정
```

설정 명령어를 실행하면 다음 정보를 입력받아 자동으로 config.json을 생성합니다:
- Jira 사이트 URL (예: https://your-site.atlassian.net)
- 프로젝트 키 (예: PROJ)
- 이메일
- API 토큰 ([Atlassian API 토큰 생성](https://id.atlassian.com/manage-profile/security/api-tokens))

## 주요 명령어

### 조회

| 명령어 | 설명 |
|--------|------|
| `/jira` | 현재 브랜치의 티켓 조회 |
| `/jira 조회 AS-1234` | 특정 티켓 조회 |

### 목록

| 명령어 | 설명 |
|--------|------|
| `/jira 목록` | 내 담당 티켓 목록 |
| `/jira 목록 --담당자 홍길동` | 특정 담당자 티켓 |
| `/jira 목록 --빈티켓` | 설명 없는 티켓 |
| `/jira 목록 --상태 진행중` | 상태별 필터링 |
| `/jira 목록 --스프린트` | 현재 스프린트 이슈 |
| `/jira 목록 --스프린트 "Sprint 8"` | 특정 스프린트 이슈 |
| `/jira 목록 --백로그` | 백로그 이슈 |
| `/jira 스프린트` | 스프린트 목록 조회 |

### 상태 변경

| 명령어 | 설명 |
|--------|------|
| `/jira 상태 진행중` | 현재 티켓 상태 변경 |
| `/jira 상태 AS-1234 완료` | 특정 티켓 상태 변경 |

상태 매핑: `할일/todo` → To Do, `진행중/시작` → In Progress, `리뷰/검토` → In Review, `완료/done` → Done

### 할당

| 명령어 | 설명 |
|--------|------|
| `/jira 할당 홍길동` | 현재 티켓 담당자 변경 |
| `/jira 할당 AS-1234 홍길동` | 특정 티켓 담당자 변경 |

### 코멘트

| 명령어 | 설명 |
|--------|------|
| `/jira 코멘트 "내용"` | 현재 티켓에 코멘트 추가 |
| `/jira 코멘트 AS-1234 "내용"` | 특정 티켓에 코멘트 추가 |

### 서브태스크 및 필드

| 명령어 | 설명 |
|--------|------|
| `/jira 서브태스크 AS-1234 "제목"` | 하위 작업 생성 |
| `/jira 추정 AS-1234 2h` | 추정치 설정 (1h, 2d, 1w) |

### 브랜치

| 명령어 | 설명 |
|--------|------|
| `/jira 브랜치 AS-1234 설명` | 티켓 기반 브랜치 생성 |

브랜치 prefix는 이슈 타입에 따라 자동 결정: `feature/`, `bugfix/`, `hotfix/`, `refactor/`

### 작성 및 설정

| 명령어 | 설명 |
|--------|------|
| `/jira 작성` | 티켓 내용 작성 (대화형) |
| `/jira 설정` | 환경 설정 조회 (대화형) |
| `/jira 설정 --보드` | 보드 목록 조회 |
| `/jira 설정 --필드` | 필드 목록 조회 |

## 브랜치 형식

스킬은 브랜치명에서 티켓 번호를 자동 추출합니다:

```
feature/AS-1234/short-description
bugfix/AS-5678
AS-9999
```

## 요구사항

- Node.js 18+
- npx tsx (TypeScript 실행)
- Jira Cloud 계정 및 API 토큰
